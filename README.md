# Project 1: Data Foundations (Titanic Dataset)

This repository contains the implementation of the Data Science foundations project.

## Overview
1. **Data Collection & Dataset Understanding**: Loading data and basic statistics.
2. **Data Cleaning & Preprocessing**: Handling null values and dropping unnecessary columns.
3. **Exploratory Data Analysis (EDA)**: Data visualizations analyzing survival rates.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Open `task1_notebook.ipynb` in your Jupyter environment.
3. Run all cells.
